# AI-Based Flower Image Classification using TensorFlow and CNN

## Project Description

This project uses Artificial Intelligence and Deep Learning to classify flower images into different categories such as Daisy, Dandelion, Rose, Sunflower, and Tulip using TensorFlow and Convolutional Neural Networks (CNN).

## Technologies Used

* Python
* TensorFlow
* NumPy
* Matplotlib

## Dataset

* Flowers Recognition Dataset from Kaggle

## Classes

* Daisy
* Dandelion
* Rose
* Sunflower
* Tulip

## How to Run

1. Run `train.py` to train the model.
2. Run `predict.py` to predict the flower class for a new image.
